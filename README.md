# Code and Erratum for Presentations for Categories of Crystals

This repository contains code and supplementary files for the paper **Presentations for Categories of Crystals** by David He and Daniel Tubbenhauer.

The paper gives generators and relations for monoidal categories of crystals generated by fundamental crystals. The code in this repository is intended to produce and check the finite lists of generators and relations appearing in the paper, especially in small rank and in the cases where the list is too long to be printed comfortably in the paper. In particular, the type `G_2` output is meant to live here rather than in the article. This is academia's version of tidying up: move the mess somewhere reproducible.

An erratum, if needed, can be found at the bottom of this page.

# What is in this repository?

At the moment this repository is a placeholder for the code accompanying the paper. The intended contents are:

* code to construct fundamental crystals;
* code to compute atoms;
* code to compute crossings / chosen commutors;
* code to compute the `H=I` and orientation-reversing relations;
* output files for the small-rank examples appearing in the paper;
* full output for cases, such as `G_2`, where the relation list is too long to print in the paper.

The exact file structure will be added once the code is cleaned up enough to not embarrass its authors.
